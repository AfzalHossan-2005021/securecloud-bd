# ──────────────────────────────────────────────────────────────────────────────
# requirements-data.txt
#
# Pinned dependencies for data download, EDA, and preprocessing work.
# These are a subset of ml/requirements.txt with Jupyter and visualization
# tools added.  Install into an isolated environment:
#
#   python -m venv .venv-data
#   source .venv-data/bin/activate        # Linux / macOS
#   .venv-data\Scripts\activate           # Windows
#   pip install -r ml/data/requirements-data.txt
#
# Version constraints:
#   pandas 2.2.x — required for the new Copy-on-Write default and
#                  ArrowDtype support used in the preprocessing pipelines.
#   numpy  1.26.x — last series before NumPy 2.0 ABI break; keeps
#                   compatibility with scikit-learn 1.4.x and TF 2.15.
#   scikit-learn 1.4.x — must match ml/requirements.txt exactly so that
#                   IForestDetector joblib files are portable between
#                   the data env and the training env.
# ──────────────────────────────────────────────────────────────────────────────
