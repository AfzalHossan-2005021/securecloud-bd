# datasets/

Preprocessing pipelines for the two benchmark datasets used to train and evaluate
the SecureCloud-BD anomaly-detection models.

## Datasets

### UNSW-NB15

- **Source:** https://research.unsw.edu.au/projects/unsw-nb15-dataset
- **Download:** `UNSW_NB15_training-set.csv` and `UNSW_NB15_testing-set.csv`
- **Preprocessor:** `datasets/unsw_nb15/preprocess.py`

```bash
python datasets/unsw_nb15/preprocess.py \
  --train path/to/UNSW_NB15_training-set.csv \
  --test  path/to/UNSW_NB15_testing-set.csv \
  --out   datasets/unsw_nb15/processed/
```

### CIC-IDS2017

- **Source:** https://www.unb.ca/cic/datasets/ids-2017.html
- **Download:** All CSVs from the `MachineLearningCVE/` folder
- **Preprocessor:** `datasets/cic_ids2017/preprocess.py`

```bash
python datasets/cic_ids2017/preprocess.py \
  --input-dir path/to/MachineLearningCVE/ \
  --out        datasets/cic_ids2017/processed/
```

## Output schema

Both pipelines emit Parquet files with the same **20-feature canonical schema**
plus a binary `label` column (0 = normal, 1 = attack):

| # | Feature | Source mapping |
|---|---------|---------------|
| 1 | `duration` | `dur` / `Flow Duration` |
| 2 | `orig_bytes` | `sbytes` / `Total Fwd Packets` |
| 3 | `resp_bytes` | `dbytes` / `Total Bwd Packets` |
| 4–7 | packet/IP byte counts | see preprocessors |
| 8 | `missed_bytes` | set to 0 for CIC (not available) |
| 9–11 | `proto_tcp/udp/icmp` | one-hot from `proto` |
| 12–15 | `conn_state_*` | one-hot from `state` |
| 16–18 | `service_http/dns/ssl` | one-hot from `service` |
| 19–20 | `bytes_per_pkt_orig/resp` | derived |

MinMax scaling is fit on the training split and re-used on the test split.

## Directory structure

```
datasets/
├── unsw_nb15/
│   ├── preprocess.py
│   └── processed/        ← gitignored; generated by preprocess.py
│       ├── train.parquet
│       └── test.parquet
├── cic_ids2017/
│   ├── preprocess.py
│   └── processed/        ← gitignored
│       ├── train.parquet
│       └── test.parquet
└── tests/
    └── test_preprocess.py
```

Raw CSVs are **not** committed to this repository.
